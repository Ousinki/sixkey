# Chrome Web Store Listing

## Name
SixKey Braille – 盲文悬浮注音

## Short Description (≤132 chars)
Hover over Chinese text to see braille annotations (现行盲文). Shows initials, finals, tones, and dot patterns in real time.

## Full Description
SixKey Braille adds real-time braille annotations to any Chinese text on the web.

Simply hover your mouse over Chinese characters, and a popup instantly displays:
• Braille dot patterns (现行盲文 standard, GB/T 15720)
• Pinyin with initial/final breakdown
• Dot position numbers
• SVG visual dot matrix

Features:
✦ Hover Annotation — Move your mouse over Chinese text to see braille annotations
✦ Tone Marks — Optional tone markers (声调) for pronunciation clarity
✦ Dot Numbers — Show/hide braille dot position numbers
✦ English Braille — Toggle international braille for English words (Grade 1)
✦ Chinese Punctuation — Full support for Chinese punctuation braille mapping
✦ Braille Converter — Interactive tool to convert between Chinese text and braille
✦ Reference Table — Complete lookup table for initials, finals, tones, and English letters
✦ Dark Mode — Eye-friendly dark theme
✦ Zero Network — All processing happens locally, no data sent anywhere

Perfect for:
• Students learning Chinese braille (现行盲文)
• Braille transcribers and teachers
• Accessibility professionals
• Anyone curious about how Chinese braille works

This extension does not collect any user data. See our privacy policy for details.
