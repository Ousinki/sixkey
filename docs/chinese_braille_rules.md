# 现行盲文规则总结

> 来源：《基础盲文》第二章 汉语盲文 · 第一节 现行盲文（第 20–24 页）

现行盲文的内容主要包括：**字母符形**、**字词书写规则**、**标点符号书写规则**和**分词连写规则**。其中，字母符形是基础，字母符形和字词书写规则涉及盲文字词本身的书写，标点符号和分词连写规则涉及字词基础上的语句书写。

---

## 一、字母符形

汉语盲文都是拼音制的文字形式，书写的基本上都是汉字的音节。在拼写时，根据汉语音节结构，采用**声、韵双拼制**，即一个盲字一般由**声母**和**韵母**两方符形构成。因此，盲文书写中涉及的汉语拼音中的声母和韵母都有其对应的盲文符形。

当前，盲文书写时涉及的字母共有 **56 个**：
- **声母 21 个** → 对应的盲文符形 **18 个**
- **韵母 36 个**（含 1 个特殊韵母 er）→ 对应的盲文符形 **34 个**

### 识记技巧

1. **点位连线法**：将字母符形中的点位进行连线以构成特定的几何形状，进行形象记忆。例如：声母 l 的符形中 1、2、3 点连线后，就是一条垂直线「|」，与声母 l 的形状几乎相同。
2. **编顺口溜法**：将几个字母放在一起识记。例如将六个点位中只缺一个点的字母放在一起，按缺点的点位顺序进行排列（ue 缺 1 点、uai 缺 2 点、uan 缺 3 点、ou 缺 4 点、üan 缺 5 点），根据字母读音编顺口溜「月歪晚欧元」来识记。
3. **找关系法**：构建两个或更多个字母或符形之间的某种关系（如对称、有规律变化），通过这种关系来实现高效识记。例如 p 与 iong 符形之间，可视作存在对称关系。四个韵母 an、ang、en、eng 的符形整体上可视作左右对称且有规律变化的关系，甚至可以视作一个「⏣」几何图形，以此来同时识记四个符形。

## 二、字词拼写规则

利用声母、韵母符形拼写盲文字词时，需要遵循盲文字词拼写规则。拼写规则包括**字**的拼写规则，也包括**词**的拼写规则。

### 规则 1：声母 + 韵母构成一个盲字

一般情况下，一个盲字由**声母、韵母二方符形**构成，**声母在前，韵母在后**。

- 例：家 jiā → 声母 j + 韵母 ia
- 例：妈 mā → 声母 m + 韵母 a
- 例：华 huá → 声母 h + 韵母 ua

### 规则 2：多字词连续拼写，字与字之间不空方

由两个及以上盲字构成的词语，按字的先后**依次连续拼写**，字与字之间不空方。

- 例：家庭 jiātíng → j+ia+t+ing（四个符形连写）
- 例：妈妈 māma → m+a+m+a
- 例：电风扇 diànfēngshàn → d+ian+f+eng+sh+an

### 规则 3：韵母可以单独构成盲字

如果一个字（音节）与某个韵母的读音相同，那么这个**韵母可以单独构成此盲字**。

- 例：乌鸦 wūyā → 韵母 u + 韵母 ia（"乌"的读音等同于韵母 u，"鸦"的读音等同于韵母 ia）
- 例：论文 lùnwén → l+un+韵母 en（"文"的读音等同于韵母 en 加声调）

### 规则 4：zhi、chi、shi、ri、zi、ci、si 只写声母

音节为 **zhi、chi、shi、ri、zi、ci、si** 的盲字，拼写时**只写出声母** zh、ch、sh、r、z、c、s 的符形即可，**不写韵母 i**。

- 例：吃饭 chīfàn → ch+f+an（"吃"只写声母 ch）
- 例：知识 zhīshi → zh+sh（"知"只写 zh，"识"只写 sh）

## 三、标调规则

声调是汉语音节的重要组成部分。在盲文中，四个声调都有其对应的符形：

| 声调 | 调号 | 说明 |
|:---:|:---:|:---:|
| 阴平（一声） | ˉ | 第一声，如 mā |
| 阳平（二声） | ´ | 第二声，如 má |
| 上声（三声） | ˇ | 第三声，如 mǎ |
| 去声（四声） | ` | 第四声，如 mà |

在现行盲文中，对声调的书写有一定的规则要求。规则包括以下五点：

### 规则 1：声调符形写在盲字后面

盲字标调时，声调符形写在**需要标调的盲字后面**。

- 例：天空 tiānkōng → t+ian+阴平+k+ong+阴平
- 例：学生 xuésheng → x+üe+阳平+sh+eng（"生"为轻声，没有对应的符形）
- 例：特殊 tèshū → t+e+去声+sh+u+阴平

### 规则 2：一般情况下不标声调

书写现行盲文时，**一般情况下不标声调**。字词的读音往往依据上下文来确定。

- 例：新中国的首都是北京（不标调）
- 例：天气寒冷，睡觉需要盖上厚被子（不标调）

### 规则 3：声母独用 + 韵母独用之间必须加界音声调

在一个词里，如果前面的盲字是声母 **zh、ch、sh、r、z、c、s 单独构成的**，后面的盲字是由**韵母单独构成的**，那么两个盲字之间**必须加前字的声调符号**，用以界音，防止音节间界限发生混淆。

- 例：植物 zhíwù → zh+阳平+u+去声（"植"= zh 独用，"物"= u 韵母独用，需加界音声调）
- 例：市委 shìwěi → sh+去声+ui+上声（"市"= sh 独用，"委"= ui 韵母独用）
- 例：思维 sīwéi → s+阴平+ui+阳平（"思"= s 独用，"维"= ui 韵母独用）

### 规则 4：陌生或少见的人名、地名及字词需要标调

除了界音，**陌生或少见的人名、地名及字词**也需要使用声调来帮助发音。

- 例：王小天 wángxiǎotiān → 需标调
- 例：琚湾镇 jūwānzhèn → 需标调
- 例：饕餮 tāotiè → 需标调

### 规则 5：常用同音、近音字词的特别规定

为区别一些同音、近音字词，对一些常用字、词的声调书写情况进行了特别规定：

| 字 | 拼写说明 |
|:---:|:---|
| 他 | **不写声调** |
| 她 | **写声调**（用声调区分"他"和"她"） |
| 它 | **不写声调**，但在声母符形前加写一个四点符形，作为特殊标记 |
| 有 | **不写声调** |
| 是 | **不写声调** |
| 在 | **不写声调** |

## 四、标点符号

标点符号是书面语言的一个有机组成部分。正确使用标点符号，能帮助读者分清句子结构，辨明语气和意义。现行盲文使用了现代汉语中所有的标点符号，并根据点字符号的特点，适当增加了几个，在书写形式上也有特别的规定。

### 1. 句号（。）

⠐⠆ — (5) (2,3) — **两方符形构成。**

书写规定：
1. 句号与前、后的词语之间**均不空方**。
2. 句号**不能写在行首**。
3. 如果一行末尾不够方写句号时，须将句号前面的一个词语与句号**一起移至下一行**书写。

### 2. 问号（？）

⠐⠄ — (5) (3) — **两方符形构成。**

书写规定：
1. 问号与前、后的词语之间**均不空方**。
2. 问号**不能写在行首**。
3. 如果一行末尾不够方写问号时，须将问号前面的一个词语与问号**一起移至下一行**书写。

### 3. 感叹号（！）

⠰⠂ — (5,6) (2) — **两方符形构成。**

书写规定：
1. 感叹号与前、后的词语之间**均不空方**。
2. 感叹号**不能写在行首**。
3. 如果一行末尾不够方写感叹号时，须将感叹号前面的一个词语与感叹号**一起移至下一行**书写。

### 4. 逗号（，）

⠐ — (5) — **一方符形构成。**

书写规定：
1. 逗号与前面的词语之间**不空方**，与后面的词语之间**空一方**。
2. 逗号**不能写在行首**。
3. 如果一行末尾不够方写逗号时，须将逗号前面的一个词语与逗号移至下一行书写；如果逗号正好在一行的最末一方，逗号后文字移到下一行书写时，下一行**开头不要空方**。

### 5. 顿号（、）

⠈ — (4) — **一方符形构成。**

书写规定：
1. 顿号与前面的词语之间**不空方**，与后面的词语之间**空一方**。
2. 顿号**不能写在行首**。
3. 如果一行末尾不够方写顿号时，须将顿号前面的一个词语与顿号移至下一行书写；如果顿号正好在一行的最末一方，顿号后文字移到下一行书写时，下一行**开头不要空方**。

### 6. 冒号（：）

⠤ — (3,6) — **一方符形构成。**

书写规定：
1. 冒号与前面的词语之间**不空方**，与后面的词语之间**空一方**。
2. 冒号**不能写在行首**。
3. 如果一行末尾不够方写冒号时，须将冒号前面的一个词语与冒号移至下一行书写；如果冒号正好在一行的最末一方，冒号后文字移到下一行书写时，下一行**开头不要空方**。

### 7. 分号（；）

⠰ — (5,6) — **一方符形构成。**

书写规定：
1. 分号与前面的词语之间**不空方**，与后面的词语之间**空一方**。
2. 分号**不能写在行首**。
3. 如果一行末尾不够方写分号时，须将分号前面的一个词语与分号移至下一行书写；如果分号正好在一行的最末一方，分号后文字移到下一行书写时，下一行**开头不要空方**。

### 8. 破折号（——）

⠠⠤ — (6) (3,6) — **两方符形构成。**

书写规定：
1. 破折号与前、后的词语之间**均不空方**。
2. 破折号**可以写在行首**。
3. 构成破折号的两方符形**必须写在同一行**。
4. 如果一行末尾不够方写破折号时，须将破折号**完整地移至下一行**书写。

### 9. 省略号（……）

⠐⠐⠐ — (5) (5) (5) — **三方符形构成。**

书写规定：
1. 省略号与前面的词语之间**不空方**，与后面的词语之间**空一方**。
2. 省略号**要写在行首**。
3. 省略号**必须三方写在同一行**。
4. 如果一行末尾不够方写省略号时，须将省略号**移至下一行**书写。
5. 省略号与前面的逗号之间**空一方**。省略号与后面的标点之间**不空方**。

### 10. 引号（" "）

⠘ — (4,5) — 引号的开号和关号**都是一方符形，且同形**构成。

书写规定：
1. 引号的开号与前面的词语之间**空一方**，引号的关号与后面的词语之间**空一方**。
2. 引号的开号可以写在行首，前面不用空方，但**不能写在一行末尾**。
3. 引号的关号可以写在一行末尾，但**不能写在行首**。
4. 引号的关号在一行末尾不够方书写时，须与前面的词语**一起移至下一行**书写。
5. 引号的开号同前面占两方的标点符号之间**不空方**。

### 11. 括号（( )、[ ]）

- **圆括号（( )）**：开号 ⠰⠄ (5,6)(3)，关号 ⠠⠆ (6)(2,3)，都是**两方符形构成**。
- **方括号（[ ]）**：开号关号同形 ⠰⠆ (5,6)(2,3)，都是**两方符形构成**。

书写规定：
1. 括号的开号与前面的词语之间**不空方**，括号的关号与后面的词语之间**不空方**。
2. 括号的开号可以写在行首，但**不能写在一行末尾**。
3. 括号的关号可以写在一行末尾，但**不能写在行首**。
4. 括号的关号在一行末尾不够方书写时，须与前面的词语**一起移至下一行**书写。
5. 括号的关号同后面的标点符号之间**不空方**。

### 12. 书名号（《 》、〈 〉）

- **双书名号（《 》）**：开号 ⠐⠤ (5)(3,6)，关号 ⠤⠂ (3,6)(2)，都是**两方符形构成**。
- **单书名号（〈 〉）**：开号 ⠐⠄ (5)(3)，关号 ⠠⠂ (6)(2)，都是**两方符形构成**。

书写规定：
1. 书名号的开号与前面的词语之间**空一方**，书名号的关号与后面的词语之间**空一方**。
2. 书名号的开号可以写在行首，前面不用空方，但**不能写在一行末尾**。
3. 书名号的关号可以写在一行末尾，但**不能写在行首**。
4. 书名号的关号在一行末尾不够方书写时，须与前面的词语**一起移至下一行**书写。
5. 书名号的开号同前面占两方的标点符号之间**不空方**。

### 13. 换行连写号

⠤ — (3,6) — **一方符形。**

书写规定：
1. 在一行末尾，如果一个词没有写完，有音节需要移至下一行书写，那么在下一行的**开头要先写换行连写号**，再写没有写完的音节。
2. 一个音节中的**声母与韵母之间不能使用换行连写号**，音节与声调或标点符号之间也不能使用换行连写号。

### 14. 间隔号（·）

⠠⠄ — (6) (3) — **两方符形构成。**

书写规定：
1. 间隔号与前面的词语和后面的词语之间**都不空方**。
2. 间隔号可以写在一行末尾，但**不能写在行首**。
3. 间隔号在一行末尾不够方书写时，须与前面的词语**一起移至下一行**书写。

### 15. 着重号

⠐ — (5) — **一方符形。**

书写规定：
1. 在需要着重的词语**前面**写着重号，需要着重的词语与着重号之间**不空方**。着重号与前面的词语之间**空一方**。
2. 着重号和需要着重的词语**必须在同一行**。

### 标点符号通用注意事项

1. 由**两方或三方符形**构成的符号，**不能拆开来分放在上下两行**。
2. 前后连用的两个标点符号，如果前面的符号要求后空一方，后面的符号要求前空一方，这时，只要在两个标点符号之间**空一方即可**。

## 五、分词连写规则

### （一）分词连写的含义

分词连写是我国大陆地区汉语盲文特有的重要书写规则。分词连写指的是书写汉语盲文语句时，对语句中词语的书写方式作出规定（后续章节详述具体规则）。
